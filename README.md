# 微服务电商系统

基于Spring Cloud微服务架构搭建的电子商城

---
### 技术栈：
- 依赖管理：Maven
- ORM框架：Mybatis
- 数据库：Mysql
- 服务注册中心组件：Eureka
- JS框架：JQuery
- CSS框架：Bootstrap3
- 应用部署：Docker
- 数据库缓存：Redis
- 安全框架：SpringSecurity
- 服务调用：RestTemplate + Ribbon
