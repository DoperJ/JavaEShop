# 华工软件应用系统课程设计
基于SpringBoot搭建，用到一些SpringCloud的技术实现微服务架构

---
### 技术栈：
- 依赖管理：Maven
- ORM框架：Mybatis
- 数据库：Mysql
- 服务注册中心组件：Eureka
- JS框架：JQuery
- CSS框架：Bootstrap3
- 应用部署：Docker
- 数据库缓存：Redis
- 安全框架：SpringSecurity
- API网关：Zuul
- 服务调用：RestTemplate + Ribbon
